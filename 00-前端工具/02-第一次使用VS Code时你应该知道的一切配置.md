## 常用快捷键


### 编辑


**删除文字**：


| 快捷键 | 作用 | 备注 |
|:-------------|:-----|:-----|
|delete|删除光标的前一个字符|相当于 Windows 键盘上的退格键|
|fn + delete|删除光标的后一个字符||
|**option + delete**|删除光标之前的一个单词|英文有效|
|**command + delete**|删除光标之前的整行内容|【荐】|
